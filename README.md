# Whatsapp-Chatbot-Backend

This is a simple backend project built using Java and Spring Boot.

## Features

* Accepts a message using POST API
* Returns a reply based on input
* Simple chatbot logic using if-else

## API Endpoint

POST /webhook

## Sample Request

{
"message": "Hi"
}

## Sample Response

{
"reply": "Hello"
}

## Tech Used

* Java
* Spring Boot
